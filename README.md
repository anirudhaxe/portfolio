# Portfolio

A modern personal portfolio website built with Astro, TypeScript, and Tailwind CSS. Features a Neovim mode.

---

## Tech Stack

| Technology       | Purpose                                           |
| ---------------- | ------------------------------------------------- |
| **Astro**        | The web framework for building the website        |
| **TypeScript**   | For type safety and improved developer experience |
| **Tailwind CSS** | For styling the website                           |
| **React**        | Used for some interactive components              |
| **Vercel**       | For deployment                                    |

---

## Getting Started

Follow these steps to get a local copy up and running.

### Prerequisites

- **Node.js** (v22.x or later)
- **pnpm** package manager

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/anirudhaxe/portfolio.git
   ```

2. **Install dependencies**
   ```bash
   pnpm install
   ```

### Running the Development Server

Start the development server with:

```bash
pnpm run dev
```

The server will be available at `http://localhost:4321`.
